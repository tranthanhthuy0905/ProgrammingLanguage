HISTORY
1. What is the name of your language?
C++ (C Plus Plus) 

2. When/where was it created, and by whom? Was it written to address a particular problem or need?
C++ is a middle-level language created by Bjarne Stroustrup as a development in C family in 1985. 
The language was born when the author realized lots of limitations of the current languages such as 
the slowness of Simula, the low-level factor of BCPL, etc. He took advantage of his experience in 
working with C language during his PhD education to develop C++ basement. It emerged with the purpose
of maximizing the performance, the efficiency and flexibility of use of system and software resources.

GETTING STARTED
1. Install VSCode
2. Install C/C++, Code Runner extensions
3. Run the program: Right Click => Run code
Running success check: A HelloWord.exe file in the folder appears
4. Stop the program: Right Click => Stop code

*** Tips: ***
To print out the program's result to the terminal instead of output:
- Open Settings, search for 'Code Runner', check on "Whether to run code in Integrated Terminal".
